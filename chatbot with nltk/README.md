this is a demo chatbot written in python with its nltk package.

NLTK :
natural language toolkit also known as nltk is a suit created in python programming language for the process of natural language proceesing.
this suite contains libraires and programs that supports classification, tokenization, stemming, tagging, parsing, and semantic functionalities. 

To create our chat bot, we are using the nltk.chat module in nltk library. 

imports :

from nltk.chat.util import Chat , reflections

chat - This is a class that has all the logic that is used by the chatbot.

Reflections - this is a dictionary. it contains a set of input and corresponding out put values.this act as an optional dictionary for the chatbot program. 
eg: "i am" : "you are"
